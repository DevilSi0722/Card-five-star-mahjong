# 卡五星麻将

卡五星麻将是一款可以直接在浏览器里玩的 3D 麻将游戏。游戏以三人卡五星为核心玩法，使用筒、条和中发白牌，支持单人练习和好友联机对战。

## 你可以玩什么

- 单人模式：进入后立即开局，与两位电脑对手对战。
- 多人模式：创建 4 位房间号，邀请好友加入；不足三人时可由电脑自动补位。
- 3D 牌桌：掷骰、发牌、手牌、弃牌区和碰杠牌组都会在牌桌上展示。
- 自动提示：轮到你时会显示可用的碰、杠、胡、过、亮倒等操作。
- 听牌信息：上听或亮倒后会展示可胡牌、剩余张数和倍率。
- 局内音效：摸牌、弃牌、掷骰等动作带有音效，可随时开关。
- 快捷聊天：多人房间内可以发送预设聊天短句。
- 移动端适配：手机横屏时会切换为更紧凑的牌桌和操作布局。

## 开始游戏

打开游戏后会先进入大厅。

选择“单人模式”会直接开始一局，对手由电脑控制。适合熟悉规则、练习出牌和测试番型。

选择“多人模式”后，可以创建或加入房间。创建房间时可以设置总局数、底分、是否开启“亮倒自摸买马”，也可以随机生成或自定义 4 位房间号。加入房间时输入好友给你的 4 位房间号即可。

在等待室里，每位玩家可以选择风位。房主至少需要等待 2 名真人玩家进入后才能开始游戏；如果人数不足三人，空位会在开始时自动补电脑。

## 基本操作

- 出牌：点击手牌选中，再次点击同一张牌打出；手机横屏时也可以上滑打出。
- 碰、杠、胡：当你可以响应他人出牌或自己摸牌可操作时，右侧/底部会出现对应按钮。
- 亮倒：听牌时可以点击“亮”，再选择要打出的牌完成亮倒。
- 过：不想执行当前响应时，点击“过”。
- 新局/准备：单人结算后可直接开新局；多人结算后点击“准备”，所有真人玩家准备后自动进入下一局。
- 退出：局内设置区可以返回大厅或离开房间。

## 玩法要点

本游戏是三人卡五星玩法，牌面包含：

- 筒子：1 筒到 9 筒
- 条子：1 条到 9 条
- 字牌：中、发、白

游戏支持的主要动作包括出牌、碰、明杠、暗杠、补杠、抢杠胡、自摸、点炮胡和亮倒。

胡牌结算会展示每位玩家本局得分、总分、番型与杠分明细。当前支持的番型包括基础胡、碰碰胡、明四归一、暗四归一、七对、龙七对、双龙七对、大三元、小三元、清一色、卡五星、手抓一、亮倒/明牌、杠上开花、杠上炮、抢杠胡和海底捞。

如果开启“亮倒自摸买马”，亮倒后自摸胡牌会触发买马奖励，结算时会单独展示买马结果。

## 界面功能

牌桌左上角会显示牌墙剩余张数、音效开关、设置和退出按钮。右上角会显示三位玩家的当前分数，并高亮当前行动玩家。

桌面端会在顶部显示对局日志，方便回看最近的摸打、碰杠、胡牌和流局信息。多人模式下，左右两侧会显示其他玩家昵称、风位和分数。

设置中可以调整桌布和是否启用物理碰撞弃牌。单人模式还可以设置下一局底分和是否开启亮倒自摸买马；多人模式的局数、底分和买马开关由房主创建房间时决定，局内不可修改。

## 多人联机说明

多人模式需要部署者配置 Firebase 联机服务。如果大厅中的“多人模式”显示不可用，说明当前站点还没有配置联机服务；此时仍然可以正常游玩单人模式。

联机时房主负责推进牌局，其他玩家通过房间同步自己的视角。房间号可以点击复制，分享给好友后即可加入同一局。

## 本地体验

如果你是在自己的电脑上打开这个项目，可以用下面的命令启动：

```bash
npm install
npm run dev
```

启动后访问：

```text
http://localhost:11450
```

单人模式不需要额外配置。多人模式需要先配置 Firebase 环境变量，变量名可以参考 `.env.local.example`。
